## 🎮 How to Use (Detailed)

### Adding Labels:
```
1. Fill in form fields:
   • Product Name (e.g., "Jeans")
   • SKU (optional, e.g., "JNS-001")
   • Price (e.g., "1299")
   • Quantity (1-27, default 27)

2. Click "Add Item"

3. Label appears in list on left
   Label preview appears on right

4. Repeat for more products (same sheet)
```

### Removing Labels:
```
1. Hover over item in "Items to Print" list
2. Red delete icon appears
3. Click to remove
4. Preview updates automatically
```

### Printing Labels:
```
1. Ensure all labels are correct
2. Check right panel preview
3. Load label sheet in printer
4. Click "Print Price Tags"
5. Print dialog opens
6. Select printer and settings
7. Click "Print"
8. Wait for printing to complete
9. Remove label sheet from printer
```

## ❓ Common Questions

**Q: Can I print fewer than 27 labels?**
A: Yes! Set quantity to 1-26, remaining spaces will be blank.

**Q: Can I print more than 27 labels?**
A: Add multiple items. Each gets its own quantity allocation (max 27 total).

**Q: What if I don't have special label sheets?**
A: You can print on regular A4 paper and cut manually, but professional labels work better.

**Q: Do my labels need to be connected to products in inventory?**
A: No, they're independent. But you can manually match them to inventory items.

**Q: Will my labels save if I close the browser?**
A: Yes! They automatically save to your browser. Clearing browser data will delete them.

**Q: Can I print from my phone?**
A: Yes, but for best results use a computer with access to your label printer.

**Q: What if the printer isn't detected?**
A: Refresh the page, ensure printer is connected, and try again.

**Q: Can I change the label design?**
A: The design is currently fixed for consistency. Contact support for custom designs.
